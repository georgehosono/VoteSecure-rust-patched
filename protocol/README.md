# VoteSecure Protocol Library

Actor implementations and data structures required for implementation of the VoteSecure protocol.

## Overview

This library contains the actors and data structures necessary to implement the VoteSecure protocol, including both Internet-facing and air-gapped (trustee-related) components. It includes actors to be used in the following implementations:

- Election Administration Server
- Digital Ballot Box
- Voter Application
- Ballot Check Application
- Trustee Administration Server
- Trustee Application

It also includes interfaces that must be implemented by:

- Authentication Server
- Public Bulletin Board
- Storage Layer for Digital Ballot Box
